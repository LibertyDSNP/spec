# DSNP Over Frequency

## Language and Framework
Frequency is a Polkadot Parachain written in Rust, using the Substrate framework.

- [Frequency Website](https://www.frequency.xyz)
- [Frequency GitHub](https://github.com/LibertyDSNP/frequency)
