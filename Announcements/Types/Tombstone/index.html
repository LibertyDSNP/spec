Tombstone Announcement

A Tombstone Announcement is a way to note that a previous announcement signature is invalid and the related Announcement should be considered reverted. It is NOT possible to revert a tombstone.

Field Requirements

announcementType

  • MUST be fixed to 0

createdAt

  • MUST be set to the milliseconds since Unix epoch at time of signing

fromId

  • MUST be a DSNP User Id
  • MUST be the signer of the target announcement

targetAnnouncementType

  • MUST be the Announcement Type of the target announcement
  • MUST ONLY be a Tombstone Allowed Announcement Type

  • MUST be the signature of the fromId user
